# ASFEnhance

> 增强ASF的功能

## 可用命令

| 命令              | 说明                                         |
| ----------------- | -------------------------------------------- |
| K                 | 从文本提取key                                |
| KEY               | 同上                                         |
| P \[botName\]     | 获取机器人的点数余额                         |
| POINT \[botName\] | 同上                                         |
| PA                | 等同于`POINT ASF`·`,获取所有机器人的点数余额 |
| LA                | 等同于`LEVEL ASF`,获取所有机器人的等级       |
| BA                | 等同于`BALANCE ASF`,获取所有机器人的钱包余额 |
| BA                | 等同于`BALANCE ASF`,获取所有机器人的钱包余额 |
